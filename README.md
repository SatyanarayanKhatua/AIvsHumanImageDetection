# AI vs Human Image Detection

## 📌 Overview

AI vs Human Image Detection is a deep learning-based web application that classifies whether an uploaded image is AI-generated or captured by a human. The project uses a pre-trained VGG19 convolutional neural network with transfer learning to achieve high classification accuracy. A simple web interface allows users to upload images and receive instant predictions.

---

## ✨ Features

- Detects AI-generated and human-generated images
- Upload image through a user-friendly interface
- Real-time prediction
- Deep learning model using VGG19
- Image preprocessing before prediction
- Responsive web interface

## 🚀 Installation



